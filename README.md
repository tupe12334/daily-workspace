# Daily Workspace

A personal Claude Code workspace configured for daily work summaries, productivity tracking, and engineering workflows.

## Setup

1. Copy `info.example.md` to `info.md` and fill in your details:

   ```yaml
   work_email: "your@email.com"
   github_username: "your-username"
   github_organization: "your-org"
   slack_user_id: "UXXXXXXXX"
   ```

## Usage

Open Claude Code in this directory and use any skill directly:

```
/work-daily
/work-github yesterday
```
